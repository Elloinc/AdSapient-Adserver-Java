Setting up a Campaign

In this tutorial we will teach you how to set up a campaign.

1. First of all you need to login to an account with advertising priviledges (advertiser, advertiser+publisher, admin). If you have not got one, you can register on the login screen.
2. Click on 'Campaigns' and then on 'Add'. Fill in the required fields and set the campaign's settings.
3. From the campaign's settings (click on 'Edit' in the campaigns' list) add a new creative to this campaign. Choose the creative's type and upload its file.
4. Set the campaign's and banner's status to active from their settings.
