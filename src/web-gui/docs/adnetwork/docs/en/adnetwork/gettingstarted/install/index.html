Installing AdSapient Overview

1. Preinstall JBoss and MySQL (also see System Requirements).

2. Create database named 'adsapient' in MySQL.

3. Create tables and propagate them with data using sql files in the 'schems' folder of the war file that we send to you.

4. Deploy war file (read war deployment instructions for JBoss).

5. Change hibernate.properties in the root of the deployed war file and add banner-ds.xml to the deployment folder of JBoss previously changing it to fit your mysql connection.

6. Open your browser and type the address where JBoss is started. The default application context path is /adsapient.
