About AdSapient

AdSapient is a full-fledged 3rd generation ad server capable of serving unlimited number of any type of creative to any publishing areas whether it is a video clip displayed specifically to users with high bandwidth connection or a small ad code loaded into an html email.

Main features:
- multiuser ad management system with user-friendly web-based interface;
- 3rd generation ad serving functionality includes advanced targeting and campaign setup;
- detailed reporting;
- supports all media formats;
- content/traffic/time/geo/ targeting;
- various system roles: advertiser, publisher, advertiser+publisher, admin;
- revenue-oriented system design and logic.
