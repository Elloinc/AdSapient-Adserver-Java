Traffic Filter

The traffic filter allows to fine-tune the number of scheduled impressions and clicks for the campaign.

It has the following self-explanatory settings:
1. Scheduled number of impressions for the whole campaign.
2. Scheduled number of clicks for the whole campaign.
3. Scheduled daily number of delivered impressions.
4. Scheduled daily number of delivered clicks.
5. Impression frequency cap per unique user:
- daily;
- monthly;
- for the whole campaign;
- for a selected period
6. Click frequency cap per unique user:
- daily;
- monthly;
- for the whole campaign;
- for a selected period.
