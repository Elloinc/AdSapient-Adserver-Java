Campaign Settings

Each campaign has the following settings:
ID - unique campaign identifier set by the system;
Campaign Name - campaign name set by the user;
Start Date - date at which the campaign should be started (its status will be automatically set to active);
End Date - date at which the campaign will be ended (its status will be set to inactive);
Target URL - advertised URL where the banners from this campaign will redirect to;
Status - status of the campaign: active or inactive. All banners within the campaign comply with the status of the holder campaign.

Campaign's settings also contain the list of banners created within the campaign.
