Banner Sizes

Banner size does not have a restricting function. It is used only for matching ad place's size with the creative's size, so that the creative is not distorted when served to a browser. The name of a banner size does not necessarily have to be of the NNxNN format. You can name it whatever you like. But for usability reasons it is recommended to use width and height of the banner size in its name.
