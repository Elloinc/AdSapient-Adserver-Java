Using Targeting Templates

First of all you have to create a targeting template. Click on the link 'Targeting Templates' and then on 'Add'. Browse through different targeting settings, click on 'Update' to change the templates settings. Finally click on the 'Add New Template' button to create the template.
Now when in targeting dialog of a campaign you can see the list of targeting templates to select from. Choose any number of targeting templates and click on 'Apply templates'. Now the templates' settings are copied into the campaign's targeting settings.

Important: application of a template's settings has narrowing function, meaning that if the targeting settings of a campaign already have some narrowing (e.g. selected categories) then the newly applied templates will not widen the targeted selection, so the selected categories will not be unselected but more narrowing settings will be added to the campaign's targeting in case they exist in the applied template.

Note: changing the targeting settings of a campaign after a template has been applied to it does not change the settings of this template. To change the settings of a template you have to click on 'Edit' from the list of targeting templates.
