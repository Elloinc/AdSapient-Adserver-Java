Getting Started with AdSapient Banner Manager

AdSapient Banner Manager comes in two flavours: as a hosted service and as a licensed program installed on a client's server.

Hosted Solution
- comes with full service free support (the cost of support is actually included into the price for hosted ad serving)
- free regular software updates
- data backups
- 99.9% uptime
- load-sharing and failover

Software Solution
- one-time license fee
- any configuration you like

Both hosted and software solutions are based on the same software. Check the full list of AdSapient's features to learn what it can do.
