Using AdSapient Banner Manager

AdSapient has an intuitive design and interface. Most of the dialogs are self-explanatory.
The user roles and their functions within the system are described below.

Admininstrator (also called trafficker)
- defines system settings like categories, page positions, default banners;
- handles user accounts, can login any account and change any settings;
- tracks overall publishing and advertising reports.

Advertiser
- runs advertising campaigns;
- uses targeting options to maximize campaigns' effectiveness;
- sets campaign's budget and tracks campaign's delivery.

Publisher
- registers sites and ad places and assigns specific settings to them;
- tracks publishing and payroll reports.
