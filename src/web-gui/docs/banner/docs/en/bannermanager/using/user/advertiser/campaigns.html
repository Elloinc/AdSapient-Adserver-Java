Campaigns

The list of campaigns contains advertising campaigns created on the current account.

Quick reports in the campaigns list include:
Views/Clicks - number of ad views / clicks generated by the campaign during the delivery period;
CTR - click-through rate (The average number of click-throughs per hundred ad impressions, expressed as a percentage.)

Possible actions on campaigns:
Targeting - initiates targeting dialog for the campaign;
Reports - displays reports for the campaign;
Edit - opens campaign's settings with the option to modify and update them;
Delete - removes the campaign with all its banners completely from the system.
