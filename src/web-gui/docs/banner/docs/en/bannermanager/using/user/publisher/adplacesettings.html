Ad Place Settings

Each ad place has the following settings:
ID - unique identifier assigned to the ad place by the system;
Site ID - site ID this ad place belongs to;
Position - page position where this ad place is located;
Dimensions - banner size this ad place can load;
Category - ad place category, primarily copied from the site's category but can be set to any other category;
Banner HTML Code - HTML code which should be copied and pasted into the corresponding place on the site.
