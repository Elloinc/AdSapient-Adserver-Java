Banners

The list of banners contains banners created on the current account.

Quick reports in the banners list include:
Views/Clicks - number of ad views / clicks generated by the banner during the delivery period;
CTR - click-through rate

Possible actions on banners:
Reports - displays reports for the banner;
Edit - opens banner's settings;
Delete - removes the banner completely from the system.
