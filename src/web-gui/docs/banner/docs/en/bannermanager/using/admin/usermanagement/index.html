User Management Overview

User management is conducted by the admin from the admin interface. Click on 'All Users' link to see the list of all registered users.

The information on each user in the list includes:
ID - user account identifier assigned by the system. You can use it to indicate which user you mean since the value of the ID field is unique.
User Name - user name or company name. Do not confuse with login.
User Role - role of the user which he chose during registration. The user role cannot be changed after an account has been created.
Login - login used by this user.
Password - password is not hidden in this list.
Email - email of the user. It is used for notifications and varified during the registration process.

The actions which can be carried out on each user are:
Edit - initiates user profile dialog which allows to edit user details.
Remove - removes the user information completely from the system. All system entities (campaigns, banners, sites, ad places, targeting templates) registered on this user account will be removed respectively.
Login - opens a new browser window with the interface of the corresponding user logged in.
